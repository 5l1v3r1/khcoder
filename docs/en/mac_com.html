Preface

On Mac, you can run KH Coder with its source code. Though it's free of charges, you have to go through the highly complicated setup process in this way.

So, as a fee-based support service, I provide automatic setup software of KH Coder for $36 USD. With this software, you can easily start using KH Coder 3 or KH Coder 2 on your Mac.

*Though KH Coder itself is free of charges, this setup software is not a part of KH Coder and it is a commercial software.

Before buying it, please read following instructions carefully.

[back to the page top]

Prerequisites

1. macOS 10.6 to 10.13

2. Please download and install Xquartz if you don't have it yet.

3. Please download and install JDK (Java SE Development Kit) if you don't have it yet.

4. Restart your Mac.
